## Overview
This lab covers:
- Setting up and Eslint Rules
- compiling and transpiling the code into one file using Webpack.
- Creating Unit tests for the application using Jest.
- Creating E2E tests using Cypress.

## Instructions
The assignment instructions are split into the following pages to make it easier to follow.
1. Accept the assignment on Canvas.
2. Clone down the Repo. 
3. Install the dependencies using `npm install`
    * This installs some dependencies that you'll need later on in the assisgnment as well as others that we're not really addressing in this lab.

4. Follow the instructions for the following 4 parts:
    - [Part 1: Linting your code using `ESLint`](./linting)
    - [Part 2: Transpile your code using `Webpack`](./transpile)
    - [Part 3: Unit Testing your code using `Jest`](./unit-testing)
    - [Part 4: E2E Testing your code using `cypress`](./e2e-testing)
