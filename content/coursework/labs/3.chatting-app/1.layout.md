## Footer - Application Chat input
1. Inside the `<footer>`, create a [Bootstrapâ€™s Input groups](https://getbootstrap.com/docs/4.5/components/input-group/#button-addons) of a text input and a button.
    * give the containing `<div>` an id of `chatbar`
    * give the input field an id of `my-message`
    * give the button an id of `send-button`

    ```html
    <footer>
        <div id="chatbar" class="input-group mb-3">
            <input
                type="text"
                id="my-message"
                class="form-control"
                placeholder="Type a message ...">
            <div class="input-group-append">
                <button
                    class="btn btn-outline-primary"
                    id="send-button"
                    type="button">Send</button>
            </div>
        </div>
    </footer>
    ```
2. copy the following css code to your `styles.css` file

    ```css
    body,
    html {
        height: 100vh;
        overflow: hidden;
        margin: 0;
        padding: 0;
    }

    body {
        display: flex;
        flex-direction: column;
    }

    header,
        footer {
        background: #e9ecef;
    }

    .jumbotron {
        margin-bottom:0;
    }

    footer {
        padding: 0.5rem
    }

    #chat {
        flex: 1 0 auto;
        height: 0;
        overflow-y: auto;
        padding: 1rem;
    }
    ```
